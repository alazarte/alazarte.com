Uploader

To upload pictures or plaint text files from sendfile. The methods are:

GET /
index.html

POST /
Accept: multipart/form-data
Body format:
  When sending paste:
    title=filename.txt
    paste=multiline text file
    secret=currentSecret
  When sending picture:
    file=image/jpeg file
    secret=currentSecret

Source code: https://git.alazarte.com/cgit.cgi/uploader/
